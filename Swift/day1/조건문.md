> Swift를 처음 다루면서 가장 먼저 익숙해지고 싶은 부분이 조건문이다.앱의 UI나 기능은 대부분 "어떤 상태일 때는 이렇게, 아닐 때는 이렇게"라는 흐름으로 움직이기 때문에 조건 표현 방식이 자연스러워야 한다.간단한 예로 다크모드 여부를 체크하는 로직을 떠올리면서 Swift에서 조건을 어떻게 표현하는지 정리해본다.

---

### 1. Boolean 값과 기본적인 if 문

다크모드 여부를 하나의 상태 값으로 두고 시작했다. Swift에서는 `true` 또는 `false` 값을 가지는 변수를 Boolean 타입으로 표현한다.

```swift
var isDarkMode = true   // 다크모드 여부를 저장하는 변수

```

이 값을 기준으로 조건문을 만들어 보았다.

```swift
if isDarkMode {
    print("다크모드입니다")  // isDarkMode가 true일 때 실행
} else {
    print("다크모드가 아닙니다")  // isDarkMode가 false일 때 실행
}

```

Playground에서 실행하면 조건에 맞는 print만 출력된다.

---

### 2. 조건을 반대로 체크하는 방법

조건이 **아닐 때**를 표현할 때는 느낌표(`!`)를 붙이면 된다.

```swift
if !isDarkMode {
    print("다크모드가 아닙니다")   // isDarkMode가 false일 때 실행
}

```

`!isDarkMode`는 `isDarkMode == false`와 같은 의미다.

Swift에서는 비교식을 생략하고 Boolean 값만 넣는 방식이 더 자연스럽다.

---

### 3. 비교 연산 생략

Swift는 Boolean 변수 자체를 그대로 조건으로 사용하는 것이 허용되기 때문에 다음 두 표현은 완전히 동일하다.

```swift
if isDarkMode { ... }

```

```swift
if isDarkMode == true { ... }

```

반대로 false 비교도 마찬가지다.

```swift
if isDarkMode == false { ... }

```

```swift
if !isDarkMode { ... }

```

---

### 4. 삼항 연산자(조건 ? 값1 : 값2)

조건을 통해 값을 바로 선택해서 변수에 담고 싶을 때 삼항 연산자를 사용한다.

```swift
let title = isDarkMode ? "다크모드입니다" : "다크모드가 아닙니다"
// isDarkMode가 true → 앞의 문자열 선택
// isDarkMode가 false → 뒤의 문자열 선택

print(title)

```

비교식을 생략한 형태도 자연스럽게 사용 가능하다.

반대로 다크모드가 아닐 때 문구를 선택하고 싶다면 느낌표를 쓰면 된다.

```swift
let title = !isDarkMode ? "다크모드가 아닙니다" : "다크모드입니다"

```

삼항 연산자는 짧은 조건 분기에서 특히 효율적이다.

---

## 요약

- Boolean 변수를 기반으로 조건문을 구성한다.
- `if`, `else`, `!조건` 은 Swift 조건문에서 기본적인 흐름을 만든다.
- Boolean 비교는 생략 가능하며 `if isDarkMode {}` 형태가 가장 자연스럽다.
- 삼항 연산자는 한 줄에서 조건에 따른 값을 선택할 때 유용하다.
- 전체 흐름은 조건식 → true일 때 실행 → false일 때 실행으로 단순하지만, Swift에서는 이를 매우 간결하게 표현할 수 있다.
